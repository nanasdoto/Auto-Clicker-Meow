# Privacy Policy — Auto Clicker Meow

**Last Updated:** July 10, 2026

## Introduction

Auto Clicker Meow ("the Extension") is a browser automation tool that allows users to record, replay, and automate mouse clicks and keyboard actions. This Privacy Policy describes how the Extension handles user data.

## Data Collection

**Auto Clicker Meow does NOT collect, transmit, or share any personal data.**

Specifically, we do **not** collect:

- Personal information (name, email, address, etc.)
- Browsing history or activity
- Cookies or tracking identifiers
- IP addresses or geolocation data
- Financial or payment information
- Authentication credentials

## Data Storage

All data generated by the Extension (such as recorded macros and user preferences) is stored **locally** on your device using Chrome's built-in `chrome.storage.local` API. This data:

- Never leaves your browser
- Is never transmitted to any external server
- Is never shared with any third party
- Is fully under your control and can be deleted at any time

## Permissions

The Extension requests the following browser permissions:

| Permission | Purpose |
|------------|---------|
| `activeTab` | To interact with the currently active tab for recording and replaying actions |
| `storage` | To save your recordings and preferences locally on your device |
| `scripting` | To inject the content script that performs recording and replay on web pages |
| `tabs` | To identify the active tab for sending recording/replay commands |

These permissions are used **solely** for the Extension's core functionality and are **never** used to collect or transmit any user data.

## Network Activity

Auto Clicker Meow makes **zero network requests**. The Extension:

- Does not communicate with any external servers
- Does not use analytics or tracking services
- Does not load external resources
- Functions entirely offline after installation

## Third-Party Services

The Extension does **not** integrate with or send data to any third-party services, APIs, or analytics platforms.

## Children's Privacy

The Extension does not knowingly collect any data from children under the age of 13, as it does not collect data from any users.

## User Control

You have full control over your data:

- **View recordings:** Open the Extension popup to see all saved recordings
- **Export recordings:** Use the Export feature to save your recordings as a JSON file
- **Delete recordings:** Delete individual recordings or clear all data at any time
- **Uninstall:** Removing the Extension from Chrome will automatically delete all locally stored data

## Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the "Last Updated" date above. Continued use of the Extension after changes constitutes acceptance of the updated policy.
