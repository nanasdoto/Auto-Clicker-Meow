# AutoClicker Pro: Advanced Chrome Automation Extension

A powerful, intuitive, and lightweight Google Chrome extension designed to automate repetitive browser tasks. Whether you need a simple macro clicker, a form-filler, or a complex automation workflow with conditional logic, this extension handles it all entirely within your browser.

## 🚀 Features

### 1. Macro Recorder (Mouse & Keyboard)

- **Action Tracking:** Record your precise mouse clicks and keyboard keystrokes in real-time.
- **Visual Timeline:** View and manage your recorded actions inside a clean, structured list before replaying.

### 2. Flexible Replay Engine

- **Infinite Loops:** Run your automation continuously without interruption.
- **Manual Boundaries:** Set a specific number of repetitions (e.g., repeat exactly 50 times) to maintain total control over your workflow.

### 3. Instant Auto-Clicker (Cursor-Based)

- **Quick Trigger:** Instantly click on your current cursor position.
- **Customizable Actions:** Choose between **Left Click**, **Right Click**, or **Double Click** to match any web interface requirement.

### 4. Advanced Automation Logic (If/Else Rules)

- **Conditional Workflows:** Add smart `if/else` rules inside your recordings.
- **Dynamic Decision Making:** Make the macro behave differently based on specific web conditions (e.g., _If button 'Next' is visible, click it; else, refresh the page_).

### 5. Seamless Profile Management

- **Import & Export:** Save your complex recording flows as JSON files to backup or share them with others.
- **Duplicate:** Clone existing recordings with a single click to easily tweak and test variations without starting from scratch.

## 🛠️ Installation

Since this extension is currently in development, you can install it locally via Developer Mode:

1. Download or clone this repository:
   ```bash
   git clone [https://github.com/nanasdoto/clickflow.git](https://github.com/nanasdoto/clickflow.git)
   ```
2. Open Google Chrome and navigate to chrome://extensions/.
3. Enable Developer mode by toggling the switch in the top-right corner.
4. Click on the Load unpacked button in the top-left corner.
5. Select the root folder of this project (the folder containing the manifest.json file).
